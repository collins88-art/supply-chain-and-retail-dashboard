# supply chain and retail dashboard

## Project Overview

This project presents a comprehensive Business Intelligence solution built with Power BI to analyze business operations across sales, customers, inventory, procurement, and logistics.

The dashboards provide interactive insights that help management monitor performance, identify operational challenges, and support data-driven decision-making.

## Project Highlights

* Sales and profitability analysis
* Customer behavior and payment insights
* Inventory and stock monitoring
* Supplier and procurement performance tracking
* Shipment and delivery analysis

## Some of the business questions this project answered

* How profitable is the business?
* Which products and regions perform best?
* Which customers contribute most to revenue?
* Are inventory levels sufficient?
* Which suppliers and shipment regions experience delays?

## the tool the i  used for this project

* Power BI (power query , data modeling , dax )
  

## Key Outcomes

The analysis helped identify:

* Revenue trends and high-performing categories
* Customer purchasing patterns
* Inventory risks and stock movement
* Supplier delivery inefficiencies
* Logistics and shipment delays
